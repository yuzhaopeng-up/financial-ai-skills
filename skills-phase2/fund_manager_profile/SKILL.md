# SKILL.md - 基金经理画像技能

## 1. 技能概述

**技能名称**：fund_manager_profile（基金经理画像）

**核心能力**：快速查询并生成中国知名公募基金经理的完整画像报告，支持多维度分析（从业背景/投资风格/业绩表现/持仓偏好/风险特征/获奖记录等）

**适用场景**：
- 基金产品尽调与筛选
- 基金经理研究与评价
- 基金推荐与资产配置建议
- 路演材料制作
- 投资决策支持

---

## 2. 技能接口

### 2.1 核心引擎：FundManagerEngine

**类文件**：`manager_engine.py`

**初始化**：
```python
from manager_engine import FundManagerEngine
engine = FundManagerEngine()
```

**主要方法**：

| 方法 | 参数 | 返回 | 说明 |
|------|------|------|------|
| `get_profile(name, format='text')` | 基金经理姓名，格式(text/markdown/json) | 画像报告文本 | 获取基金经理完整画像 |
| `list_managers()` | 无 | 基金经理名称列表 | 列出所有支持的基金经理 |
| `search_manager(name)` | 基金经理姓名（支持模糊） | 匹配到的姓名或None | 模糊搜索基金经理 |
| `get_summary_table()` | 无 | Markdown汇总表 | 获取所有基金经理汇总对比表 |

**快捷函数**：
```python
from manager_engine import get_profile, list_managers
profile = get_profile("张坤")
managers = list_managers()
```

## 4. 画像维度说明

| 维度 | 说明 |
|------|------|
| **从业背景** | 入职年限、首只基金年份、管理规模、学历背景 |
| **投资风格** | 风格标签（大盘成长/价值/均衡/中小盘）、特色标签、能力圈行业 |
| **持仓偏好** | 前十大重仓股、行业集中度、个股集中度、平均持仓周期 |
| **业绩表现** | 5年年化收益、5年累计收益、最大回撤、夏普比率 |
| **风险特征** | 波动率水平、回撤控制能力、风格稳定度 |
| **获奖记录** | 晨星奖、金牛奖、明星基金奖、金基金奖 |
| **投资理念** | 经典语录 |
| **人物点评** | 性格特点与投资风格定性 |

## 5. CLI 使用方式

### 5.1 命令行入口
```bash
python3 scripts/manager_cli.py generate "基金经理画像 张坤"
python3 scripts/manager_cli.py generate "经理分析 朱少醒"
python3 scripts/manager_cli.py list
python3 scripts/manager_cli.py summary
python3 scripts/manager_cli.py search "张坤"
```

### 5.2 参数说明
| 子命令 | 说明 |
|--------|------|
| `generate <query>` | 生成基金经理画像（支持"基金经理画像 XXX"或"经理分析 XXX"格式）|
| `list` | 列出所有内置基金经理 |
| `summary` | 输出基金经理汇总对比表 |
| `search <name>` | 搜索并确认基金经理匹配结果 |

## 6. 企微卡片集成

**文件**：`wecom_integration.py`

生成适合企业微信消息卡片的格式输出，用于企微机器人推送。

```python
from wecom_integration import format_wecom_card
card = format_wecom_card("张坤")
```

企微卡片字段：
- 基金经理姓名 + 所属公司
- 管理规模 + 从业年限
- 投资风格标签
- 5年年化收益
- 回撤控制评级
- 代表基金名称
