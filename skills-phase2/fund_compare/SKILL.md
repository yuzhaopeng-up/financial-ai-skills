# 基金对比技能 (fund_compare)

## 简介

基金对比技能用于对 2-4 只基金进行多维度对比分析，帮助投资者快速了解基金之间的差异，辅助投资决策。

## 核心功能

- **多维度对比**：区间收益、风险指标、基金基本信息、基金经理、资产配置
- **智能解析**：支持基金代码（如 XXXXXX）或基金名称（如 某基金公司中小盘）混合输入
- **综合推荐**：基于量化打分给出综合评价和差异分析
- **企微卡片**：支持输出企微消息卡片格式

## 对比维度

### 1. 区间收益
| 维度 | 说明 |
|------|------|
| 近1月 | 最近1个月收益率 |
| 近3月 | 最近3个月收益率 |
| 近6月 | 最近6个月收益率 |
| 近1年 | 最近1年收益率 |
| 近3年 | 最近3年收益率 |

### 2. 风险指标
| 维度 | 说明 |
|------|------|
| 夏普比率 | 承担每单位风险获得的超额收益，越高越好 |
| 最大回撤 | 历史最大亏损幅度，越小越好 |
| 波动率 | 收益率的标准差，越低越稳健 |
| 卡玛比率 | 年化收益/最大回撤，越高越好 |

### 3. 基本信息
| 维度 | 说明 |
|------|------|
| 基金类型 | 股票型/混合型/债券型/指数型等 |
| 基金规模 | 最新净资产规模 |
| 成立时间 | 基金成立日期 |
| 管理费率 | 年度管理费 |

### 4. 基金经理
| 维度 | 说明 |
|------|------|
| 基金经理 | 在任基金经理姓名 |
| 任职时间 | 基金经理管理该基金的时长 |
| 从业年限 | 基金经理总从业年限 |

### 5. 资产配置
| 维度 | 说明 |
|------|------|
| 股票仓位 | 股票占净资产比例 |
| 债券仓位 | 债券占净资产比例 |
| 现金仓位 | 现金占净资产比例 |
| 前十大持仓 | 重仓股票/债券明细 |

## 使用方式

### CLI 方式

```bash
# 基金对比（代码）
python3 scripts/compare_cli.py generate "基金对比 XXXXXX 163402"

# 基金对比（名称）
python3 scripts/compare_cli.py generate "比较 某基金公司中小盘 兴全趋势"

# 企微卡片格式
python3 scripts/compare_cli.py generate "基金对比 XXXXXX 163402" --format wecom
```

### Python API

```python
from fund_compare import FundCompareEngine

engine = FundCompareEngine()
result = engine.compare(["XXXXXX", "163402"])
print(result["summary"])
```
