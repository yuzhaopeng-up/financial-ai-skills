# 供应链金融方案引擎 v1.0

> 核心企业 + 供应商信息 → 融资方案 + 风险控制

## 一、核心能力

| 能力 | 说明 |
|------|------|
| 模式识别 | 自动识别供应链模式（应收/订单/存货/预付/核心企业担保） |
| 方案设计 | 生成定制化融资方案 |
| 风险评估 | 多维度风险评分与控制措施 |
| 额度测算 | 供应商可融资额度估算 |

## 二、支持供应链模式（≥3种）

1. **应收账款融资** - 供应商以对核心企业的应收账款转让/质押获取融资
2. **订单融资** - 供应商以有效订单合同获取预融资
3. **存货融资** - 供应商以库存商品作为抵质押物获取融资
4. **预付款融资** - 核心企业担保，金融机构为供应商提供预付款支持
5. **核心企业担保模式** - 核心企业为链上供应商提供连带责任担保

## 三、输入格式

```
融资方案 核心企业:XXX 供应商:XXX 交易规模:XXX万 年限:X年
风险评估 核心企业:XXX 供应商:XXX 融资模式:应收账款
额度测算 供应商:XXX 年营收:XXX万 交易历史:X月
```

## 四、核心参数

- 核心企业信用评级（AAA/AA/A/BBB/BB/B）
- 供应商信用评级
- 交易规模与账期
- 历史合作月数
- 担保方式

## 五、风险控制措施

| 等级 | 措施 |
|------|------|
| 核心企业增信 | 要求核心企业确权/担保 |
| 资金闭环 | 定向支付给核心企业 |
| 存货监管 | 第三方仓储监管 |
| 交易验证 | 发票/合同交叉核验 |
| 保险覆盖 | 信用保险/财产险 |
