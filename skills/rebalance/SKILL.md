# 资产配置再平衡引擎 v1.0

> 当前持仓 + 目标配置 → 调仓方案 + 交易成本估算

## 一、核心能力

| 能力 | 说明 |
|------|------|
| 偏离度分析 | 计算各资产当前配置与目标的偏离 |
| 智能调仓 | 生成买入/卖出指令，自动平衡买卖总额 |
| 成本估算 | 佣金、印花税、滑点、杂费逐项计算 |
| 风险指标 | 最大偏离、平均偏离、HHI集中度指数 |
| 综合评估 | 方案质量评分 + 操作建议 |

## 二、支持资产类型

| 类型 | 交易成本说明 |
|------|-------------|
| 股票 | 佣金万三 + 印花税千一(卖出) + 过户费万0.2 + 滑点 |
| 债券 | 佣金万二 + 低滑点 |
| 基金/ETF | 申购费千一，无印花税 |
| 货币基金 | 零费用 |
| 黄金 | 交易费千六 + 双向滑点 |
| 外汇 | 点差/手续费千一 + 滑点 |
| 银行理财 | 零费用 |

## 三、输入格式

```
再平衡 [当前持仓] [目标配置]
```

**格式示例：**
```
再平衡 股票:400000,债券:200000,黄金:100000 股票:35,债券:25,黄金:10
```

**JSON 格式（CLI）：**
```bash
python rebalance_cli.py --holdings '{"股票":400000,"债券":200000}' --target '{"股票":50,"债券":50}'
```

## 四、输出说明

### 偏离度分析
| 指标 | 说明 |
|------|------|
| 当前% | 当前资产占总市值比例 |
| 目标% | 目标配置比例 |
| 偏离% | 当前 - 目标（正=超配，负=欠配） |

### 调仓方案
| 字段 | 说明 |
|------|------|
| 操作 | 买入/卖出 |
| 金额 | 交易金额（元） |
| 原因 | 当前占比 → 目标占比 |

### 交易成本
| 费用项 | 说明 |
|--------|------|
| 佣金 | 按费率计算，有最低值 |
| 印花税 | 仅股票卖出时收取 |
| 滑点 | 预期滑点损耗 |
| 杂费 | 过户费等 |

## 五、配置参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| tolerance | 5% | 容忍偏差，超过才触发调仓 |
| min_trade_amount | 1000元 | 最小交易金额，低于此值忽略 |

## 六、风险指标

| 指标 | 计算方式 | 解读 |
|------|---------|------|
| max_deviation | 各资产最大偏离度 | 越高越需要调仓 |
| avg_deviation | 偏离度均值 | 整体偏离程度 |
| weighted_deviation | 按目标权重加权的偏离 | 综合偏离水平 |
| HHI | Herfindahl-Hirschman指数 | 集中度风险，值越高越集中 |

## 七、验收标准

✅ 支持多资产类型：股票、债券、基金、黄金、外汇、理财、现金
✅ 输入：当前持仓（金额）+ 目标配置（百分比）
✅ 输出：调仓方案（买入/卖出明细）+ 交易成本估算
✅ 规则引擎实现，无LLM依赖
✅ 提供CLI和企微集成入口
