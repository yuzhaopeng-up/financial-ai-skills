# 并购方案生成引擎 v1.0

> 收购方 + 被收购方 + 交易目的 → 交易结构 + 估值分析 + 风险提示

## 一、核心能力

| 能力 | 说明 |
|------|------|
| 交易结构设计 | 提供多种结构方案（现金/股票/混合） |
| 估值分析 | DCF/PE/PB/PS多维估值 |
| 财务预测 | 3-5年盈利预测与协同效应测算 |
| 风险提示 | 整合监管审查要点与交易风险 |
| 方案对比 | 多方案优劣势分析 |

## 二、支持交易结构

- 现金收购（100% / 控股 / 参股）
- 股票对价（换股合并 / 定向增发）
- 混合支付（现金+股票）
- 资产置换
- 承债式收购
- 杠杆收购（LBO/MBO）

## 三、输入格式

```
并购方案 [收购方] [被收购方] [交易目的]
方案分析 [标的] [交易结构类型]
估值测算 [标的] [估值方法]
```

## 四、输出内容

1. **交易结构设计** - 交易方式、支付对价、锁定期安排
2. **估值分析** - 多方法估值结果与价格区间
3. **财务预测** - 标的未来3-5年业绩预测
4. **风险提示** - 监管审查要点、交易风险、整合风险
5. **方案对比** - 多方案优劣势与推荐建议

## 五、风险等级

| 等级 | 标识 | 说明 |
|------|------|------|
| 高风险 | 🔴 | 监管重大障碍或重大不利因素 |
| 中风险 | 🟡 | 需关注或进一步核查 |
| 低风险 | 🟢 | 常规事项或已有解决方案 |
