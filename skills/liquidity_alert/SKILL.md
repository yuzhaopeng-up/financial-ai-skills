# 资金流动性风险预警引擎 v1.0

> 资金流水 + 历史数据 → 30天流动性预测 + 预警报告

## 一、核心能力

| 能力 | 说明 |
|------|------|
| 30天流动性预测 | 基于历史模式和规则预测未来30天流动性缺口 |
| 多级预警 | 🔴红色/🟡黄色/🟢绿色三级预警机制 |
| LCR监控 | 流动性覆盖率（≥100%为达标） |
| 现金流分类 | 稳定现金流/波动现金流/季节性模式识别 |
| 压力测试 | 设定情景下的流动性缺口评估 |

## 二、输入格式

```
流动性预警 [日期] [账户余额] [流入] [流出]
现金流分析 [账户] [开始日期] [结束日期]
LCR检查 [机构名称]
流动性压力测试 [情景] [持续天数]
```

## 三、预警等级

| 等级 | 标识 | 流动性覆盖率 | 说明 |
|------|------|------------|------|
| 红色预警 | 🔴 | < 80% | 流动性严重不足，需立即处置 |
| 黄色预警 | 🟡 | 80%-100% | 流动性偏紧，需密切关注 |
| 绿色正常 | 🟢 | ≥ 100% | 流动性充足 |

## 四、预测方法（规则引擎）

- **移动平均法**: 基于过去N天平均现金流预测
- **趋势外推法**: 基于线性趋势预测未来流入/流出
- **季节性调整**: 识别月度/季度周期性波动
- **压力情景**: 设定极端流出假设

## 五、预警准确率

- 历史数据验证准确率: ≥ 85%
- 规则引擎纯计算，无LLM依赖
