# 客户健康度热力图引擎 v1.0

> 客户群ID → 热力图 + 预警名单

## 一、核心能力

| 能力 | 说明 |
|------|------|
| 多维评分 | 基于活动、互动、付款、服务、稳定性、增长6个维度评分 |
| 热力图可视化 | 文本热力图展示客户群健康度分布 |
| 预警名单 | 自动识别高风险客户并生成预警 |
| 客群分层 | 将客户分为高/中/低健康度群体 |

## 二、健康度维度

| 维度 | 权重 | 指标说明 |
|------|------|----------|
| 活动度 | 20% | 登录频次、交易频率、最近活跃时间 |
| 互动度 | 15% | 客服交互频次、服务满意度、响应率 |
| 付款度 | 25% | 账单结算及时性、逾期天数、信用额度使用率 |
| 服务度 | 15% | 投诉次数、问题解决率、满意度评分 |
| 稳定度 | 15% | 客户在册时长、留存率、产品粘性 |
| 增长度 | 10% | AUM变化、产品覆盖数、资产增长率 |

## 三、健康度等级

| 等级 | 分值区间 | 标识 | 说明 |
|------|----------|------|------|
| 高健康 | 80-100 | 🟢 | 优质客户，活跃稳定 |
| 中健康 | 60-79 | 🟡 | 正常客户，关注变化 |
| 低健康 | 40-59 | 🟠 | 预警客户，需主动维护 |
| 高风险 | 0-39 | 🔴 | 重点关注，立即介入 |

## 四、输入格式

```
客户健康度 [客户群ID]
健康度热力图 [客户群ID]
客户预警 [客户群ID]
```

## 六、输出内容

- 热力图：客户群健康度矩阵分布
- 预警名单：高风险客户清单及风险原因
- 统计摘要：各维度平均得分、等级分布
- 详细报告：每位客户的分项得分与风险点
