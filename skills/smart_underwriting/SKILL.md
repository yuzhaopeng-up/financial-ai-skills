# Smart Underwriting 智能核保引擎

## 概述

智能核保引擎，基于投保人信息自动评估风险等级，输出核保决定（标准体/次标体/拒保）、自动分级（优选/标准/加费/拒保）、保费调整系数及核保建议。

## 输入参数

| 字段 | 类型 | 说明 |
|------|------|------|
| age | int | 年龄（18-65） |
| occupation | str | 职业（企业主/公务员/程序员/建筑工人/矿工/医生等） |
| health_status | str | 健康状况（优秀/良好/一般/较差） |
| family_medical_history | str | 家族病史（无/心脑血管疾病/糖尿病/恶性肿瘤/多种） |
| annual_income | float | 年收入（万元） |
| coverage_amount | float | 保额需求（万元） |
| coverage_period | int | 保障期限（年） |
| product_type | str | 产品类型（寿险/重疾险/医疗险/意外险） |

## 健康告知规则

### 血压评估
- 正常：收缩压<120 且 舒张压<80 → 系数1.0
- 正常高值：收缩压120-129 或 舒张压80-84 → 系数1.1
- 轻度高血压：收缩压130-139 或 舒张压85-89 → 系数1.3
- 中度高血压：收缩压140-159 或 舒张压90-99 → 系数1.5
- 重度高血压：收缩压≥160 或 舒张压≥100 → 拒保

### 血糖评估
- 正常：空腹血糖<5.6 → 系数1.0
- 偏高：空腹血糖5.6-6.9 → 系数1.2
- 糖尿病：空腹血糖≥7.0 或 糖化血红蛋白≥6.5% → 拒保（寿险/重疾）

### BMI评估
- 偏瘦：BMI<18.5 → 系数1.0
- 正常：BMI 18.5-23.9 → 系数1.0
- 超重：BMI 24-27.9 → 系数1.1
- 肥胖：BMI≥28 → 系数1.3

### 既往症自动评估
- 无既往症 → 系数1.0
- 轻微既往症（已治愈、无后遗症） → 系数1.2
- 中度既往症（有影响但稳定） → 系数1.5
- 严重既往症（慢性病、需持续治疗） → 拒保

## 职业风险等级

| 职业类别 | 典型职业 | 风险系数 |
|----------|----------|----------|
| 1类 | 办公室职员、公务员、教师 | 1.0 |
| 2类 | 销售、餐饮服务、农林牧渔 | 1.1 |
| 3类 | 司机、机械操作工、电工 | 1.2 |
| 4类 | 建筑工人、消防员、警察 | 1.5 |
| 5类 | 矿工、隧道工、爆破工 | 2.0 |
| 6类 | 高空作业、潜水员、特技演员 | 拒保 |

## 拒保规则

### 绝对拒保
- 恶性肿瘤病史（治愈5年内仍拒保）
- 严重心脑血管疾病（心梗、脑中风）
- 肝硬化、肾功能衰竭
- 精神病、癫痫
- 职业类别6类（意外险除外）
- 重度高血压、重度糖尿病

### 财务核保规则
- 寿险保额/年收入 > 20倍 → 高保额需体检/财务核保
- 保额/保费比异常高 → 逆选择风险

## 输出结果

```json
{
  "underwriting_decision": "标准体|加费|除外|拒保",
  "risk_level": "优选体|标准体|次标体|拒保体",
  "premium_factor": 1.0,
  "total_risk_score": 1.25,
  "underwriting_notes": "核保建议说明",
  "rejection_reasons": ["拒保原因列表"]
}
```

## 分级标准

- **优选体**：综合系数<1.1，标准保费
- **标准体**：综合系数1.1-1.3，标准保费或轻微加费
- **次标体**：综合系数1.3-2.0，加费或除外
- **拒保体**：综合系数>2.0 或 存在绝对拒保项

## 使用方式

### CLI
```bash
python3 scripts/underwriting_cli.py generate "智能核保 年龄45 职业企业主 健康良好 年收入100万 寿险500万 20年"
```

### Python API
```python
from smart_underwriting import SmartUnderwritingEngine

engine = SmartUnderwritingEngine()
result = engine.underwrite(
    age=45,
    occupation="企业主",
    health_status="良好",
    family_medical_history="无",
    annual_income=100,
    coverage_amount=500,
    coverage_period=20,
    product_type="寿险"
)
print(result)
```
