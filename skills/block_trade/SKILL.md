# 大宗交易技能 (Block Trade Analyzer)

## 概述
大宗交易分析引擎，输入股票名称/交易数量/交易价格/买方类型，自动输出折溢价率分析、市场冲击评估、流动性评价、合规提示及历史类似案例参考。

## 功能特性
- **折溢价率分析**：以当日收盘价为基准，计算折价/溢价比例
- **市场冲击估算**：基于交易量占比 × 流动性系数估算股价冲击幅度
- **流动性评估**：结合持股集中度、日均成交量综合评估变现难度
- **合规提示**：减持比例限制、锁定期、披露要求等关键规则
- **历史案例参考**：10+ 真实大宗交易案例，按行业/规模/折价率索引

## 使用方式

```bash

python3 scripts/block_cli.py generate "大宗交易 某白酒公司 100万股 价格1600元 机构转让 --closing=1850"

```

### Python API
```python
from block_trade import BlockTradeEngine

engine = BlockTradeEngine()
result = engine.analyze(
    stock_name="某股票",
    quantity=5_000_000,      # 股数
    price=10.0,               # 成交价
    buyer_type="大股东",       # 大股东 | 机构 | 私募
    closing_price=12.0,        # 当日收盘价（可选，自动取模拟值）
    daily_volume=10_000_000,  # 当日成交总量（可选）
)
```

## 买方类型说明
- **大股东**：持股5%以上的股东，转让受严格限制
- **机构**：公募基金、券商、保险等合规机构
- **私募**：私募股权基金、资产管理计划

## 合规关键规则
1. **减持比例**：大股东90天内通过集中竞价的减持不得超过总股本1%
2. **大宗门槛**：单笔成交金额不低于200万元（或单笔不低于50万股）
3. **锁定期**：大股东、董监高转让后6个月内不得反向买入
4. **披露要求**：持股5%以上股东变动需3个交易日内公告
5. **价格限制**：成交价格必须在当日涨跌幅±10%范围内（科创板±20%）

## 输出字段
- `stock_name`: 股票名称
- `quantity`: 成交数量（股）
- `price`: 成交价格（元）
- `closing_price`: 当日收盘价（元）
- `premium_discount_rate`: 折溢价率（%，负数为折价）
- `market_impact_score`: 市场冲击评分（1-10）
- `market_impact_desc`: 市场冲击描述
- `liquidity_score`: 流动性评分（1-10）
- `liquidity_desc`: 流动性评价
- `compliance_alerts`: 合规警示列表
- `similar_cases`: 相似历史案例（3个）
- `risk_level`: 风险等级（低/中/高）
