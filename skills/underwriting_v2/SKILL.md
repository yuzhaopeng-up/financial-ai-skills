# UnderwritingV2 券商两融智能核保引擎

## 概述

券商两融智能核保引擎 v2（券商信用交易专用版），基于客户信息自动评估两融准入资格，输出准入决定、额度建议、担保物折算率、风险提示及利率建议。

## 准入条件

| 条件 | 要求 | 说明 |
|------|------|------|
| 资产门槛 | ≥ 50万 | 客户总资产（不含融资负债） |
| 持股时间 | ≥ 6个月 | 在我司普通账户持股时间 |
| 风险测评 | C4 及以上 | 有效期内风险测评结果 |
| 信用记录 | 良好 | 无重大违约、违规记录 |

**资质评分**：4/4 → 可开；2-3/4 → 有条件开；< 2 → 不开

## 担保物折算率

| 资产类型 | 折算率 | 说明 |
|----------|--------|------|
| 主板股票（沪市/深市主板、中小板、创业板） | 65% | 主流担保品 |
| 科创板股票 | 50% | 波动较大，折算率较低 |
| 北交所股票 | 50% | 同科创板 |
| 信用债 | 85% | |

| 理财产品 | 70% | |

## 额度计算规则

- **最高可借金额** = min(担保证券折算价值 × 杠杆系数, 客户总资产 × 1.0)
- **杠杆系数**：信用评分≥90 → 1.5；≥80 → 1.2；≥70 → 1.0；<70 → 0.8
- **维持担保比例** = (现金 + 证券市值) / 负债总额
- **警戒线**：150% | **平仓线**：130%

## 利率建议（年化）

| 信用等级 | 融资利率 | 融券利率 | 适用场景 |
|----------|----------|----------|----------|
| 极优（≥90分） | 7.0% | 7.0% | VIP客户 |
| 优秀（80-89分） | 7.5% | 7.5% | 优质客户 |
| 良好（70-79分） | 8.5% | 8.5% | 标准客户（基准） |
| 一般（60-69分） | 9.0% | 9.0% | 需关注 |
| 较差（<60分） | 10.0% | 10.0% | 高风险 |

基准利率：年化 8.5%，信用等级不同可在 ±1.5% 范围内调整。

## 注意事项

1. 本引擎输出仅供参考，实际两融准入以券商总部审核为准
2. 科创板、创业板注册制新股折算率可能因交易所规定调整
3. ST股票折算率较低（30%），需关注流动性风险
4. 维持担保比例低于130%时将被强制平仓
5. 利率建议为参考值，实际利率以合同约定为准
