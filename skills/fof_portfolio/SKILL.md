# FOF Portfolio Skill - 基金组合技能

## 概述

FOF Portfolio Skill（基金中基金组合管理引擎）用于根据客户的风险偏好、组合规模、投资目标和持有期限，从内池 20+ 基金中筛选构建最优基金组合，提供战略资产配置（SAA）、战术资产配置（TAA）和权重优化方案。

## 核心能力

### 1. 基金筛选引擎
- **内池基金类型**：股票型、混合型、债券型、货币型、指数型、QDII
- **筛选维度**：
  - 年化收益率（近1/3/5年）
  - 夏普比率（风险调整收益）
  - 最大回撤（下行风险）
  - 基金经理稳定性（任期、变更频率）
  - 基金规模（流动性考虑）
  - 费率（持有成本）

### 2. 战略资产配置（SAA）
- 锚定长期收益目标的大类资产配置
- 基于客户风险偏好的股债比例设定
- 各类资产的长期预期收益与风险

### 3. 战术资产配置（TAA）
- 基于宏观环境的短期偏离建议
- 估值水平调整（股债相对吸引力）
- 情绪因子（资金流向、市场情绪指标）
- 政策面影响评估

### 4. 权重优化
- 基于风险预算的权重分配
- 相关性调整（降低组合波动）
- 流动性加权

## 输入参数

| 参数 | 类型 | 说明 |
|------|------|------|
| risk_preference | string | 风险偏好：保守型、稳健型、平衡型、积极型、激进型 |
| portfolio_size | string | 组合规模，如"1亿"、"5000万" |
| investment_goal | string | 投资目标：养老规划、教育金、财富增值、资产保值 |
| holding_period | string | 持有期限，如"5年"、"3年"、"长期" |

## 输出内容

1. **基金筛选方案**：从内池筛选出的基金列表及理由
2. **战略资产配置（SAA）**：大类资产比例
3. **战术调整（TAA）**：基于当前市场的调整建议
4. **权重优化方案**：各基金的推荐配置比例
5. **基金尽调要点**：每只基金的关键关注点

## 使用方式

### CLI
```bash
python3 scripts/fof_cli.py generate "FOF组合 平衡型 规模1亿 养老规划 持有5年"
```

### Python API
```python
from fof_engine import FOFEngine

engine = FOFEngine()
result = engine.generate_portfolio(
    risk_preference="平衡型",
    portfolio_size="1亿",
    investment_goal="养老规划",
    holding_period="5年"
)
print(result)
```
