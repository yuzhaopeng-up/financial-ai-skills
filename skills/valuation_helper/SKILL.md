# 估值核算辅助引擎 v1.0

> 持仓数据 + 净值日期 → 估值计算 + 异常预警

## 一、核心能力

| 能力 | 说明 |
|------|------|
| 资产估值 | 按最新净值计算各资产当前估值 |
| 收益计算 | 计算当日/累计收益与收益率 |
| 异常检测 | 检测价格异动、流动性枯竭、清盘风险等 |
| 预警提醒 | 触发阈值时输出预警信息 |

## 二、支持品种

- 股票（A股、港股、美股）
- 债券（国债、企业债、可转债）
- 基金（公募、私募、ETF、LOF）
- 期货（商品、金融期货）
- 银行存款/理财
- 现金及等价物

## 三、输入格式

```
估值核算
持仓: 股票A 1000股@10.5元, 基金B 5000份@2.3元
净值日期: 2024-01-15

或者直接传入持仓JSON:
{"positions": [{"code": "600036.SH", "name": "股票A", "volume": 1000, "cost": 10.5, "nav_date": "2024-01-15"}]}
```

## 四、预警规则

| 等级 | 标识 | 触发条件 |
|------|------|----------|
| 严重 | 🔴 | 资产价格波动超过20%/日 |
| 警告 | 🟠 | 资产价格波动超过10%/日 |
| 注意 | 🟡 | 资产价格波动超过5%/日 |
| 提示 | 🔵 | 流动性低于阈值、清盘风险等 |

## 五、准确率目标

- 估值计算准确率：≥ 99.99%
- 异常检测召回率：≥ 95%
- 误报率：< 5%
