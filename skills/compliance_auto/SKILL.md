# 业务流程合规性自动检查引擎 v1.0

> 业务类型 + 操作记录 → 合规报告 + 整改建议

## 一、核心能力

| 能力 | 说明 |
|------|------|
| 合规规则库 | 内置10+金融行业合规规则 |
| 业务类型识别 | 自动识别业务类型并匹配规则 |
| 风险等级评估 | 高/中/低三级风险评估 |
| 整改建议生成 | 提供具体的整改建议 |

## 二、支持业务类型

- 贷款业务
- 存款业务
- 理财销售
- 保险销售
- 信用卡业务
- 跨境汇款
- 开户业务
- 反洗钱检查
- 信息披露
- 客户适当性管理

## 三、输入格式

```
合规检查 [业务类型] [操作记录]
检查 业务类型 操作记录
审计 业务类型 操作记录
```

## 四、输出报告

- 合规评分（0-100）
- 风险等级（🔴高 🟡中 🟢低）
- 违规条款清单
- 整改建议
