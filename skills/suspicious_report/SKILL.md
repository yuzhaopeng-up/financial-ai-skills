# 可疑交易报告技能 (suspicious_report)

## 概述

本技能用于识别和生成可疑交易报告，符合《金融机构大额交易和可疑交易报告管理办法》（中国人民银行令〔2006〕第2号）要求。

## 功能

- 识别可疑交易特征（资金快进快出、分散归集、虚拟货币关联、虚假贸易背景、跨境资金异常等）
- 生成符合人民银行格式的可疑交易报告
- 提供建议行动

## 可疑特征库

| 特征类型 | 描述 | 阈值/规则 |
|---------|------|----------|
| 资金快进快出 | 资金在账户停留时间极短 | T+0 或 T+1 |
| 分散归集 | 资金分散转入后集中转出 | 分散账户数 ≥ 5 |
| 虚拟货币关联 | 交易对手涉及虚拟货币交易平台 | 检测关键词 |
| 虚假贸易背景 | 交易金额、频率与贸易主体不符 | 偏离行业均值 3σ |
| 跨境资金异常 | 跨境资金流动频繁且无合理用途 | 跨境笔数占比 ≥ 30% |

## 报告类型

- **大额交易报告**: 单笔或累计交易金额达到规定标准
- **可疑交易报告**: 符合可疑交易特征，需上报人民银行

## 使用方式

```bash
# 生成可疑交易报告
python3 scripts/suspicious_cli.py generate "可疑交易 某客户 累计500万 分散20个账户 快进快出"

# 生成大额交易报告
python3 scripts/suspicious_cli.py generate "大额交易 某客户 单笔300万"
```

## 输出格式

报告包含:
- 报告类型（大额/可疑）
- 可疑特征描述
- 报告内容草稿（符合人民银行格式）
- 建议行动

## 法规依据

- 《金融机构大额交易和可疑交易报告管理办法》（中国人民银行令〔2006〕第2号）
- 《金融机构客户尽职调查和客户身份资料及交易记录保存管理办法》（中国人民银行令〔2022〕第1号）
