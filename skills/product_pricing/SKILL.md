# 产品定价引擎 v1.0

> 产品类型 + 客户风险等级 + 市场利率 → 定价方案 + 敏感性分析

## 一、核心能力

| 能力 | 说明 |
|------|------|
| 定价计算 | 根据产品类型、客户风险、市场利率输出最优定价 |
| 敏感性分析 | 利率变动对定价的影响（±25/50/75/100bp） |
| 风险加成 | 根据客户风险等级调整风险溢价 |
| 竞争力对比 | 与市场基准利率对比 |

## 二、支持产品类型

- 💰 **存款**（定期、活期、结构性存款）
- 💳 **贷款**（信用贷、抵押贷、经营贷、消费贷）
- 💱 **汇兑**（结售汇、外币兑换）
- 📈 **理财**（固定收益、净值型、结构性理财）

## 三、客户风险等级

| 等级 | 标识 | 风险溢价加成 |
|------|------|-------------|
| 保守型 | 🟢 | +0.0%（最优惠利率） |
| 稳健型 | 🟡 | +0.3%~0.5% |
| 进取型 | 🔴 | +0.8%~1.5% |

## 四、输入格式

```
产品定价 [产品类型] [客户风险等级] [市场利率]
例：产品定价 贷款 稳健型 3.25%
例：存款 保守型 2.0%
```

## 五、利率敏感性分析

分析利率变动对产品定价的影响：
- 基准情景：当前市场利率
- 上行情景：+25bp、+50bp、+75bp、+100bp
- 下行情景：-25bp、-50bp、-75bp、-100bp

---
*适用场景：客户咨询、产品营销、内部定价决策*
