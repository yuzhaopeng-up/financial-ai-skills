# 智能催收策略优化引擎 v1.0

> 逾期客户信息 + 历史催收记录 → 分层催收策略 + 定制话术

## 一、核心能力

| 能力 | 说明 |
|------|------|
| 智能分层 | 基于逾期天数(M0-M5)自动定级 |
| 意愿评估 | 基于历史记录评估还款意愿(0-100分) |
| 策略生成 | 匹配催收渠道、频率与话术类型 |
| 话术生成 | 分场景生成催收话术脚本 |
| 回收率预估 | 估算策略执行后的回收率 |
| 还款计划 | 提供分期/一次性等还款方案 |

## 二、账户分层标准

| 等级 | 逾期天数 | 标签 | 说明 |
|------|---------|------|------|
| M0 | 0天 | 正常 | 无逾期 |
| M1 | 1-30天 | 关注 | 轻度逾期 |
| M2 | 31-60天 | 预警 | 中度逾期 |
| M3 | 61-90天 | 重点催收 | 重度逾期 |
| M4 | 91-180天 | 高压催收 | 严重逾期 |
| M5 | 180天+ | 呆账核销 | 基本无法回收 |

## 三、输入格式

```
催收策略 [逾期天数] [逾期金额] [客户姓名]
催收分析 逾期45天 金额58000 客户张三 历史记录...
```

## 四、验收标准

- ✅ 回收率提升 ≥ 10%（相比无策略基准）
- ✅ 支持 M0-M5 全分层
- ✅ 话术符合监管要求，不含违规词汇
