# 抵押物智能估值引擎 v1.0

> 抵押物类型 + 地址/规格 + 面积 → 估值报告 + 风险建议

## 一、核心能力

| 能力 | 说明 |
|------|------|
| 智能估值 | 基于市场数据+折旧模型，输出估值区间 |
| 风险建议 | 评估抵押物流动性/处置难度 |
| 合规检查 | 验证抵押物合法性与权属清晰度 |
| 多类型支持 | 房产/设备/车辆/土地/应收账款/股权 |

## 二、支持抵押物类型

- 房产（住宅/商业/工业）
- 设备（生产设备/医疗设备/IT设备）
- 车辆（机动车/工程车辆）
- 土地使用权/耕地
- 应收账款
- 股权/上市公司股票

## 三、输入格式

```
抵押物估值 [类型] [地址/规格] [面积]
估值 房产 北京朝阳 100平
抵押评估 设备 型号规格XXX 若干台
```

## 四、验收标准

| 抵押物类型 | 估值偏差要求 |
|------------|-------------|
| 房产 | ≤10% |
| 设备 | ≤15% |
| 车辆 | ≤12% |
| 土地 | ≤15% |
| 应收账款 | ≤10% |
| 股权 | ≤15% |

## 五、风险等级

| 等级 | 标识 | 说明 |
|------|------|------|
| 低风险 | 🟢 | 估值充足，流动性好，处置容易 |
| 中风险 | 🟡 | 估值存在不确定性，流动性一般 |
| 高风险 | 🔴 | 估值困难，流动性差，权属复杂 |
