# fraud_detection - 反欺诈检测技能

## 概述

## 核心能力

- **FraudDetectionEngine**: Python引擎类，支持直接调用
- **CLI**: 命令行入口，支持自然语言解析
- **企微集成**: 企业微信卡片消息推送

## 使用方式

### Python引擎调用

```python
from fraud_detection import FraudDetectionEngine

engine = FraudDetectionEngine()
result = engine.detect(
    amount=500000,           # 交易金额(元)
    transaction_type="转账",  # 交易类型
    counterparty="新客户",    # 对手方信息
    transaction_time="凌晨2点", # 交易时间
    account_history={...}     # 账户历史数据(可选)
)

print(result.score)          # 0-100风险评分
print(result.level)          # 风险等级
print(result.rules)          # 触发规则列表
print(result.actions)        # 建议行动
```
