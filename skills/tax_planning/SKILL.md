# Tax Planning Skill（企业税务筹划）

## 1. 技能概述

**技能名称**: tax_planning  
**适用场景**: 企业税务筹划分析  
**输入**: 企业基本信息（行业、规模、年营收、研发投入等）  
**输出**: 税务筹划方案（主要税种 + 节税空间 + 风险点 + 筹划建议）

## 2. 核心政策依据

| 优惠政策 | 适用条件 | 优惠内容 |
|---------|---------|---------|
| 小微企业所得税优惠 | 小型微利企业（年应纳税所得额≤300万、从业人数≤300人、资产总额≤5000万） | 年应纳税所得额≤100万部分，按 12.5% 计入应纳税所得额，税率 20%（实际税负 2.5%）；100-300万部分，按 25% 计入，税率 20%（实际税负 5%） |
| 研发费用加计扣除 | 制造业/科技型中小企业 | 研发费用按 120% 加计扣除（2023年起） |
| 高新技术企业优惠 | 取得高新技术企业资格证书 | 企业所得税税率 15% |
| 技术先进型服务企业 | 认定的技术先进型服务企业 | 企业所得税税率 15% |

## 3. 支持税种

- **企业所得税**：最核心，税率差异大（25%基准 / 15%高新 / 实际税负 2.5-5%小微）
- **增值税**：小规模纳税人免税额、进项税额抵扣
- **个人所得税**：股权激励、薪酬个税筹划
- **增值税附加税**：城建税、教育费附加、地方教育附加（随增值税减免）

## 4. 主要节税策略

1. **小微优惠最大化**：确保满足小微条件，合理拆分业务
2. **研发加计扣除**：规范研发费用归集，保留完整凭证
3. **高新资格维持**：保持研发投入比例≥3%，知识产权布局
4. **区域性优惠政策**：国家级高新区、经济开发区税收返还
5. **业务架构筹划**：子公司/分公司的税负对比
