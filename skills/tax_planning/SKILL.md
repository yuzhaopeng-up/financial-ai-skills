# 税务筹划方案引擎 v1.0

> 收入类型 + 资产状况 + 地区 → 税负分析 + 节税方案

## 一、核心能力

| 能力 | 说明 |
|------|------|
| 税负分析 | 计算各税种应纳税额 |
| 税种识别 | 根据收入类型识别适用税种 |
| 节税方案 | 提供合法节税建议 |
| 地区适配 | 支持不同地区税收优惠政策 |

## 二、支持税种

| 税种 | 适用场景 |
|------|----------|
| 个人所得税 | 工资薪金、劳务报酬、股息红利、财产租赁/转让 |
| 企业所得税 | 企业经营所得 |
| 增值税 | 销售货物/服务 |
| 消费税 | 高档化妆品、珠宝、烟酒等 |
| 房产税 | 房产持有/出租 |
| 契税 | 房产土地交易 |

## 三、输入格式

```
税务筹划 收入类型=工资 收入=300000 地区=北京
税务筹划 收入类型=个体工商户 收入=800000 地区=上海
税负分析 年收入=50万 资产=房产2套 地区=深圳
```

## 四、输出内容

- 税种明细与应纳税额
- 综合税负率
- 节税方案（3-5条）
- 风险提示
