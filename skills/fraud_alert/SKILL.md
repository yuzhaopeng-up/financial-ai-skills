# fraud_alert - 实时欺诈预警技能

## 概述

本技能专注**实时交易预警**场景，基于20+条预警规则对交易进行毫秒级风险评估，返回欺诈预警等级（红色/橙色/黄色）、触发规则列表、紧急处置建议及人工复核必要性。

## 与 fraud_detection 的区别

| 维度 | fraud_detection | fraud_alert |
|------|-----------------|-------------|
| 定位 | 事后综合分析 | **实时交易预警（毫秒级）** |
| 时效 | T+1分析 | **实时拦截窗口** |
| 规则侧重 | 30+复杂关联规则 | 20+高优先级单项规则 |
| 输出 | 综合评分 | 预警等级 + 紧急处置 |
| 场景 | 调查分析、合规报告 | **支付拦截、风控审核、电签复核** |

## 核心能力

- **FraudAlertEngine**: Python引擎类，毫秒级实时检测
- **CLI**: 命令行入口，支持自然语言解析
- **企微集成**: 企业微信红色预警卡片推送

## 预警等级划分

| 预警等级 | 颜色标识 | 触发条件 | 建议行动 | 人工复核 |
|----------|----------|----------|----------|----------|
| 🔴 红色预警 | 红色 | 命中A类规则 或 得分≥75 | **立即拦截**，通知风控，冻结账户 | **必须** |
| 🟠 橙色预警 | 橙色 | 命中B类规则 或 得分50-74 | **延迟放行**，人工复核通过后再放行 | **必须** |
| 🟡 黄色预警 | 黄色 | 命中C类规则 或 得分25-49 | **标记关注**，纳入监控清单 | 建议 |
| ✅ 正常 | 绿色 | 无规则触发 且 得分<25 | 正常通过 | 否 |

## 使用方式

### Python引擎调用

```python
from fraud_alert import FraudAlertEngine

engine = FraudAlertEngine()
result = engine.alert(
    amount=50000,            # 交易金额(元)
    transaction_type="转账", # 交易类型
    transaction_time="22:00", # 交易时间(HH:MM)
    location="异地",          # 地点描述
    device_change=True,      # 设备是否更换
    account_id="A12345",     # 账户ID
    history={...}            # 历史行为数据(可选)
)

print(result.level)         # 红色/橙色/黄色/正常
print(result.score)         # 0-100风险分
print(result.rules)         # 触发规则列表
print(result.actions)       # 紧急处置建议
print(result.review_required) # 是否需要人工复核
```

### CLI调用

```bash
python scripts/alert_cli.py generate "fraud_alert 交易金额5万 时间22:00 地点异地 设备更换"
python scripts/alert_cli.py alert --amount 50000 --time "22:00" --location "异地" --device-change
```

### 企微红色预警卡片

```python
from fraud_alert.wecom_integration import send_red_alert
send_red_alert(result, webhook_url="https://qyapi.weixin.qq.com/...")
```

## 输入参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| amount | float | 是 | 交易金额(元) |
| transaction_type | str | 是 | 交易类型: 转账/支付/提现/消费 |
| transaction_time | str | 否 | 交易时间(HH:MM格式或描述) |
| location | str | 否 | 地点描述(异地/同城/境外等) |
| device_change | bool | 否 | 设备是否更换/异常 |
| device_id | str | 否 | 设备指纹ID |
| account_id | str | 否 | 账户ID |
| counterparty | str | 否 | 对手方描述 |
| history | dict | 否 | 历史行为数据 |
