## 二、P0 核心场景 (立即启动)

### 2.1 市场观点生成
- **场景代码**: `market_view`
- **场景描述**: 自动生成金融市场日报/周报
- **输入**: 日期范围、行业/产品关键词
- **输出**: 结构化市场分析报告
- **预计工时**: 3天
- **依赖技能**: 
  - research_rag (已有)
  - financial-intelligence (已有)
- **验收标准**: 
  - 支持日报/周报两种格式
  - 覆盖大盘+热点+资金流向
  - 生成时间 < 30秒
- **状态**: 待开发
- **技能路径**: skills/market_view/

### 2.2 流失客户召回
- **场景代码**: `churn_recall`
- **场景描述**: 识别潜在流失客户并生成召回话术
- **输入**: 客户ID列表、流失原因
- **输出**: 客户画像 + 个性化召回话术
- **预计工时**: 5天
- **依赖技能**: 
  - customer-persona (已有)
  - customer-marketing (已有)
- **验收标准**:
  - 流失特征识别准确率 ≥ 80%
  - 召回话术满意度 ≥ 4.0/5.0
- **状态**: 待开发
- **技能路径**: skills/churn_recall/

### 2.3 客户异议训练
- **场景代码**: `objection_training`
- **场景描述**: 模拟客户异议场景进行培训
- **输入**: 产品类型、客户类型、异议类型
- **输出**: 模拟对话 + 应对话术 + 评分
- **预计工时**: 4天
- **依赖技能**: 
  - customer-persona (已有)
- **验收标准**:
  - 支持5种以上异议类型
  - 提供评分和改进建议
- **状态**: 待开发
- **技能路径**: skills/objection_training/
