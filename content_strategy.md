## 四、自动发布技术方案

### 已验证：知乎
```python
# 方案：Cookie持久化+定时发布
headers = {
    "Cookie": "z_c0=xxx",  # 从浏览器获取
    "x-requested-with": "fetch"
}
# API: /api/v4/answers 或 /api/v4/articles
```

### 待验证：掘金
```python
# 方案：模拟登录获取sessionid
# 1. POST /passport/web/login
# 2. 获取HttpOnly cookie: sessionid
# 3. POST /content_api/v1/article 发布
# 注意：sessionid有有效期，需定期刷新
```

### 待验证：CSDN
```python
# 方案：阿里云API Gateway签名
# 1. 登录获取 x-ca-key, x-ca-nonce
# 2. 每个请求需计算签名
# 3. 复杂度较高，建议先用Selenium
```

### 待验证：博客园
```python
# 方案：简单Cookie可能可行
# 1. 登录获取 .CNBlogsCookie
# 2. POST /api/posts 发布
# 测试优先级：高（可能最简单）
```

### 推荐：微信公众号
```python
# 方案1：微信测试号API（有限制）
```
