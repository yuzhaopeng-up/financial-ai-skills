## 快速开始

```bash
# 克隆仓库
git clone https://github.com/yuzhaopeng-up/financial-ai-skills.git
cd financial-ai-skills

# 安装依赖
pip install -r requirements.txt

# 运行测试
pytest tests/

# 运行示例
cd skills/financial-intelligence
python examples/cashflow_forecast.py
```

## 贡献指南

### 如何贡献

```bash
# 1. Fork仓库
# 2. 创建分支
git checkout -b feature/new-skill

# 3. 开发Skill
mkdir skills/new-skill
touch skills/new-skill/SKILL.md
touch skills/new-skill/__init__.py

# 4. 编写测试
touch tests/test_new_skill.py

# 5. 提交PR
git add .
git commit -m "feat: add new skill for xxx"
git push origin feature/new-skill
```
